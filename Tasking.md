# Tasking
## 需求解读
```
1、非特殊数字：正常输出。
2、只是1号特殊数字的倍数： Fizz。 
3、只是2号特殊数字的倍数： Buzz。 
4、只是3号特殊数字的倍数： Whizz。 
4、同为任意两个数字的倍数： FizzBuzz。 
4、同为任意三个数字的倍数： FizzBuzzWhizz。
```
## 任务划分
```
- 主逻辑                      5mins
- 倍数判断                    5mins
- 特殊数字生成                5mins
- 测试                       15mins

```
